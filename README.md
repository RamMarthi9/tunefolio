# TuneFolio

**Finance with Finesse**

TuneFolio is a read-only portfolio intelligence platform that explains
investment performance over time using secure, consent-based integration
with Zerodha.

## Why TuneFolio
- Time-aware portfolio analysis (YoY, QoQ, MoM)
- Decision attribution, not predictions
- No trading, no advice, no execution

## Product Principles
- Insight over action
- Trust over automation
- Data as a product
